# 📘 Databases and SQL: From 0 to 100

A **production-ready, end-to-end SQL learning repository** covering everything from fundamentals to advanced database concepts.


## Features

- ✅ Full PostgreSQL database setup (Dockerized)
- ✅ Real-world relational schema (`university_system`)
- ✅ Structured chapters (Basic → Advanced SQL)
- ✅ Hands-on SQL scripts for every concept
- ✅ Exercises + solutions
- ✅ Covers performance, indexing, transactions, and scaling


## 🐳 Quick Start (Docker)

### 1. Clone the repo
```bash
git clone https://github.com/samansiadati/Databases-and-SQL-From-0-to-100.git
cd Databases-and-SQL-From-0-to-100

```



### 2. Start Docker
```bash
docker-compose up -d
```
